Recurrent Phase Reconstruction Using Estimated Phase Derivatives from Deep Neural Networks | Listening Examples

All speakers are contained in the test set which means they have not been seen by the DNNs before. The clean audio files used to create the following audio examples were taken from the VCTK database [1].

The following methods were used to create the audio examples:

  • Magnitude (zero-phase): Apply inverse short-time Fourier transform (ISTFT) to the given magnitude spectrum or, if specified, apply 10 or 100 iterations of the Griffin-Lim algorithm (see [4]).
  • IF integral: Integrate the instantaneous frequency (IF) estimated by a DNN to reconstruct the phase spectrum, then combine it with the magnitude spectrum and apply the ISTFT (similar to [3]).
  • RPU: Use recurrent phase unwrapping (RPU) to reconstruct the phase spectrum from phase derivatives estimated by DNNs, then combine it with the magnitude spectrum and apply the ISTFT (see [2]).
  • Proposed: Use our weighted averaging procedure to reconstruct the phase spectrum from phase derivatives estimated by DNNs, then combine it with the magnitude spectrum and apply the ISTFT (see [5]).
